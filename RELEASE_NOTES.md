# Initial Clean Release v0.1.2

## 中文

Photo Metadata Editor v0.1.2 是一个离线 Windows EXIF/XMP/IPTC/QuickTime 元数据编辑器。本次全新发布整理了项目结构、统一作者署名、补充中英文界面和文档，并加强了备份恢复、写后验证、隐私扫描和许可证检查。

### 下载说明

- `PhotoMetadataEditor-0.1.2-win64.msi`：Windows 安装包。
- `PhotoMetaEditor-portable.zip`：免安装便携包，完整解压后运行 `PhotoMetaEditor.exe`。
- `PhotoMetaEditor.exe`：便携目录内的启动程序；单独下载时仍需要 ZIP 中相邻的 `_internal` 与 `exiftool` 文件。
- `SHA256SUMS.txt`：所有上传产物的 SHA-256 校验值。

公开产物在没有受信商业代码签名证书时保持未签名，Windows 可能显示未知发布者。

## English

Photo Metadata Editor v0.1.2 is an offline Windows editor for EXIF, XMP, IPTC, and QuickTime metadata. This clean release organizes the project, standardizes author metadata, adds complete Chinese/English UI and documentation, and strengthens backup restoration, write verification, privacy scanning, and license checks.

### Downloads

- `PhotoMetadataEditor-0.1.2-win64.msi`: Windows installer.
- `PhotoMetaEditor-portable.zip`: portable package; extract all files before running `PhotoMetaEditor.exe`.
- `PhotoMetaEditor.exe`: launcher from the portable directory; when downloaded alone it still requires adjacent `_internal` and `exiftool` files from the ZIP.
- `SHA256SUMS.txt`: SHA-256 checksums for all uploaded artifacts.

Public artifacts remain unsigned when no trusted commercial code-signing certificate is available, so Windows may display an unknown-publisher warning.
